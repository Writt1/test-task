### Параметры GET

Начало и конец времени бронирования машины должны вписываться в ссылку страницы в формате:

```
?newStart=13.11.2025%2009:40:00&newEnd=13.11.2025%2010:30:00
```


### Вывод

Результат выводится с помощью:

```
print_r($this->arResult);
```

1. В случае, если время не указано или не получилось найти должность пользователя, выводится ошибка:
   ```
   Не удалось определить должность пользователя.
   Не переданы параметры newStart или newEnd
   ```

3. В случае, если доступных машин нет:
   ```
   Array ( [NoCarsAvailable] => Нет свободных машин на это время )
   ```

4. В случае, если есть доступные машины:
   ```
   Array ( [324] => Array ( [Model] => КИА [ComfortCategory] => Третья категория [Driver] => Сергеев Сергей Сергеевич ) [331] => Array ( [Model] => Corolla [ComfortCategory] => Вторая категория [Driver] => Иванов Иван Иванович ) )
   ```
